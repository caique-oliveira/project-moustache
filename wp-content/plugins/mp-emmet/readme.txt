Upload, Activate

== Changelog ==

= 1.3.1 =

* Added the ability to choose the background color for icons in the Feature widget section.
* Added ability to add custom Google map styles.

= 1.3.0 =

* Added the ability to replace built-in contact form with any other contact form via shortcode
* Added the ability to set custom IDs for each section on the front page
* Added the ability to use icon instead of the image in Feature widget
* Made Portfolio categories separate from the blog categories

= 1.2.7 =

* Minor bugfixes and improvements

= 1.2.5 =

* New: mp_emmet_portfolio_posts_per_page filter

= 1.2.3 =

* Added the ability to set an order of the sections

= 1.2.2 =

* Fixed an issue with Team section

= 1.2.0 =

* All theme sections moved to plugin

= 1.1.0 =

* Contact section moved to plugin

= 1.0 =

* Initial version.
